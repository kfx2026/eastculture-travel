南京

漫游金陵，赴一场千年温柔旧梦

广告

核心景区

中山陵

位置：南京市玄武区紫金山南麓

历史背景：中山陵是近代中国民主革命先行者孙中山先生的陵寝，建于1926-1929年。三百九十二级石阶自下而上，掩映在苍松翠柏之间，气势恢弘。建筑融合了中国传统宫殿美学与现代纪念建筑理念，庄严肃穆，是镌刻近代风骨的精神地标。

亮点：

  • 攀392级石阶而上，每登一层，视野更开阔，陵寝愈发巍峨
  • 祭堂内肃穆的孙中山大理石坐像
  • 顶部平台俯瞰整个南京城天际线
  • 毗邻灵谷寺、明孝陵，合称"紫金山三大景区"
  • 周边梅园每年2-3月梅花盛开

开放时间：8:30–17:00（周一闭馆维护祭堂）

门票：免费（需微信小程序提前预约）

交通：地铁2号线苜蓿园站1号口出，步行15分钟或乘景区接驳车（¥10）

建议游玩：2–3小时

最佳季节：春季（3-4月梅花盛开）、秋季（10-11月层林尽染）

贴士：穿舒适的鞋——这是真爬山。上午9点前到可避开人潮。日出时分登顶的景色令人屏息。

明孝陵

位置：南京市玄武区紫金山南麓

历史背景：明孝陵是明朝开国皇帝朱元璋与马皇后的合葬陵墓，中国规模最大的帝王陵寝之一，世界文化遗产。神道蜿蜒八百余米，两侧石象生（骆驼、大象、麒麟等）静静伫立六百年，守护着这位布衣天子的安息之所。红墙石道、古木参天，静谧中透着皇家气象。其建筑布局深刻影响了北京明十三陵及东亚地区皇家陵寝的设计。

亮点：

  • 漫步神道——石象、石骆驼、石麒麟栩栩如生，堪称明代石刻艺术巅峰
  • 红墙、石牌坊、古樟树营造出悠然古朴的氛围
  • 秋季是绝佳时节：银杏"黄金大道"美不胜收
  • 四方城内的巨型龟趺石碑——"大明孝陵神功圣德碑"
  • 2003年列入世界文化遗产（明清皇家陵寝组成部分）

开放时间：7:00–18:00（3-10月）；7:00–17:30（11-2月）

门票：¥70（旺季），¥60（淡季）

交通：地铁2号线苜蓿园站3号口出，穿梅花山即到

建议游玩：2–3小时

最佳季节：深秋（10月下旬-11月）赏银杏；早春赏梅花

贴士：与中山陵联游（两个景区相邻）。神道在晨光中最上镜。

夫子庙-秦淮河风光带

位置：南京市秦淮区贡院街

历史背景：秦淮河自六朝以来就是南京的文化心脏。这里曾是士子云集、诗人唱和、商贾辐辏之地。乌衣巷青砖斑驳，朱雀桥晚风习习，唐代诗人刘禹锡笔下的"旧时王谢堂前燕，飞入寻常百姓家"至今仍是金陵最动人的注脚。夫子庙始建于宋景祐元年（1034年），是中国古代最大的科举考场江南贡院所在地。

亮点：

  • 夜泊秦淮——桨声灯影里的六朝繁华，彩灯倒映水面，恍若隔世
  • 乌衣巷——踏过唐代诗人走过的青石板路
  • 江南贡院——中国古代规模最大的科举考场
  • 朱雀桥——晚风吹拂，仿佛穿越千年
  • 秦淮灯会（春节期间）——万千花灯映照河面，延续千年民俗盛景

开放时间：夫子庙9:00–22:00；秦淮河游船18:00–21:30

门票：夫子庙¥30；游船夜票¥80、日票¥60；乌衣巷免费

交通：地铁3号线夫子庙站2号口出，步行5分钟

建议游玩：2–4小时（建议下午去，待到晚上）

最佳季节：春节期间赏灯；春秋夜晚最有意境

贴士：这一带到晚上才真正活起来——白天是旅行团，夜晚才是金陵。沿河小吃摊值得一家家试。

老门东

位置：南京市秦淮区中华门以东

历史背景：老门东是南京保存最完好的明清历史街区，位于中华门（中国规模最大的瓮城城门）以东。青石板路纵横交错，老宅院错落有致，曾世代生活着老南京人家。近年来精心修缮后，这里成为古韵与新生碰撞的文化创意街区：百年老宅里开出了独立书店、手作工坊、茶馆小院，新潮与传统在此完美交融。

亮点：

  • 穿行迷宫般的青石板小巷——每个拐角都可能遇见一方天井、一口古井
  • 非遗手作坊：金陵剪纸、绒花制作、木板水印体验
  • 露台咖啡馆，可望见古城墙与中华门
  • 夜间氛围：灯笼亮起，老砖墙在暖光下格外温柔
  • 南京印象餐厅——庭院深处的地道金陵菜

开放时间：全天开放（店铺一般10:00–21:00）

门票：免费（各店铺/工坊消费自理）

交通：地铁3号线武定门站4号口出，步行10分钟

建议游玩：2–3小时

最佳季节：全年皆宜；夜晚最有韵味

贴士：下午来，待到日落——从白昼到掌灯时分的过渡最美。找家茶馆，点一碗桂花糖芋苗。

特色美食

盐水鸭

风味特点：皮白肉嫩、肥而不腻、清淡回甘。肉质细嫩如玉，带着花椒和八角的淡淡清香。与烤鸭不同，盐水鸭冷食最佳——油脂晶莹剔透，每片都带着纯粹清淡的鲜美，"金陵第一味"名不虚传。

历史文化故事：传说盐水鸭始于六朝，历经千年打磨至臻。明代已是进贡宫廷的珍馐。如今它是南京的城市名片——本地人从家庭聚餐到节庆宴席，每周必吃。做法看似简单：鸭子用盐和香料腌制，晾干，文火浸熟——功夫全在火候与调味分寸之间。

怎么吃：

  1. 冷食，带皮切薄片
  2. 先空口品尝，感受鸭肉本真之味
  3. 也可蘸少许黑醋姜丝
  4. 配热米饭，或冷盘佐啤酒，皆宜

价格：¥40–80/整只；¥25–40/半只（餐厅）；¥15–20/外卖盒装

去哪吃：

  • 韩复兴——创于1866年，金字招牌
  • 金陵饭店——精致版本，品质稳定
  • 本地菜市场熟食档——出人意料的好，价格只要一半

鸭血粉丝汤

风味特点：鸭骨熬制的醇厚白汤是灵魂，浓郁鲜香。滑嫩鸭血（口感似豆腐）、嚼劲十足的鸭胗、软糯鸭肝，搭配爽滑红薯粉丝，在金色汤底中交融。撒一把香菜，点几滴辣油，每一勺都是金陵的烟火气息。

历史文化故事：这是南京人的终极治愈美食——大多数本地人的一天从一碗鸭血粉丝汤开始。它源于南京千年养鸭传统：物尽其用，平凡的下水被做成令人痴迷的佳肴。鸭血尤其珍贵，口感丝滑，在中医看来有补血养气之效。如今它是南京人不用言说的早餐默契——从建筑工人到公司白领，清晨排队就为这一碗。

怎么吃：

  1. 先喝汤——这是整碗的灵魂
  2. 加辣油和陈醋调味
  3. 粉丝、鸭血、鸭杂交替品尝，感受不同口感
  4. 一滴不剩喝完——碗底那口汤最浓

价格：¥15–25/碗；路边摊¥10–15

去哪吃：

  • 尹氏汤包——连锁名店，品质稳定
  • 回味——遍布全市的本地连锁，清晨即开
  • 居民区附近随便一家早餐铺——往往最地道

蟹黄汤包

风味特点：薄如纸的面皮包裹着一汪金黄蟹汤。蟹黄、蟹肉与猪肉交融，鲜甜丰腴，浓郁奢华。筷子一戳，蟹香蒸腾，满室飘散着长江的味道。

历史文化故事：南京地处江南蟹乡交汇处。每年秋季，高邮湖大闸蟹膏满黄肥之时，全城厨房都把它化为这精巧的汤包。与上海小笼包不同，南京蟹黄汤包个头更大——常一笼一只，配吸管直接喝汤。

怎么吃：

  1. 用筷子轻提汤包顶端
  2. 放入汤匙，在侧面戳一小孔
  3. 从小孔吸尽滚烫蟹汤（或用吸管）
  4. 加几滴黑醋姜丝
  5. 一两口吃掉皮和馅

价格：¥25–45/笼（6-8只）；精品单只¥15–30

去哪吃：

  • 刘长兴——夫子庙附近的百年老店
  • 狮子楼——以巨型单只蟹黄汤包著称
  • 秋季（9-11月）秦淮河沿线任意一家餐厅

民俗文化

秦淮灯会

起源与历史：秦淮灯会始于南朝（公元420-589年），鼎盛于明代南京作为帝都之时。一千五百余年来，每逢新春元宵，夫子庙灯火璀璨，万千手扎花灯沿秦淮河、挂满街巷、漂于水面，将整座城市变成灯光的海洋。秦淮灯会已被列为国家级非物质文化遗产。

体验形式：

  • 大型灯组展示：生肖、神兽、宫殿场景
  • 秦淮河夜游赏灯
  • 猜灯谜（传统互动游戏）
  • 民俗表演：南京白局、舞龙、传统音乐
  • 沿街小吃摊绵延不绝

去哪体验：

  • 夫子庙主会场
  • 老门东分会场
  • 中华门大型装置展

最佳时间：春节至元宵节期间（1月下旬-2月）；亮灯时间18:00–22:00

文化意义：秦淮灯会是南京人最珍视的民俗传统——一家人春节看灯，如同欧洲人的圣诞集市。它是六朝古都皇家灯会遗风在民间的活态传承。

南京云锦与非遗手作

起源与历史：南京是世界文学之都，非物质文化遗产宝库。最负盛名的南京云锦已有1600余年历史——丝线交织金线、孔雀羽，专供皇家龙袍。与之相伴的金陵剪纸（纤巧红纸艺术）、绒花制作（南京新娘传统头饰）、金陵刻经（木版水印佛经），共同织就了代代相传的非遗图景。

体验形式：

  • 观看云锦织工在传统木织机上操作——两人一机，一天仅织5-6厘米
  • 尝试金陵剪纸——大师几分钟就能剪出精美图案
  • 绒花工坊：学习制作南京新娘佩戴的迷你蚕丝花
  • 南京白局——用南京方言演唱的地方曲艺，软糯悠扬

去哪体验：

  • 南京云锦博物馆——现场织造演示
  • 老门东南京非遗馆
  • 甘熙故居（九十九间半）——古宅中的手作工坊

最佳时间：全年；博物馆每日开放；手作工坊周末最热闹

文化意义：这些手艺不是封存的文物，而是南京作为文化之都2600年不绝的文脉。云锦的一针一线、剪纸的一刀一划，承载着世世代代在匠心之中寻找美的人们的审美精神。

境外出行攻略

最佳出行时间

黄金季节（最宜出行）：春季（3-5月）和秋季（9-11月）。春季梅花山万梅齐放（2-3月），气温舒适（15-25°C）。秋季紫金山层林尽染，银杏金黄、枫叶火红——10月至11月被公认为南京最美的季节。

次佳季节：6月和9月初。偏热但可接受，游客少，住宿价格回落。

尽量避开：

  • 7-8月：酷暑湿热（35-38°C），南京是"四大火炉"之一
  • 国庆黄金周（10月1-7日）：景区爆满，酒店价格翻2-3倍
  • 1月下旬：湿冷刺骨（但秦淮灯会值得为此破例）

建议天数

最少：2天（紫金山景区+秦淮河一带）

理想：3天——加上侵华日军南京大屠杀遇难同胞纪念馆（沉重而肃穆）、总统府、玄武湖

深度：4-5天——南京博物院（中国最好的博物馆之一）、鸡鸣寺、栖霞山（秋色）、一天往返镇江或扬州

市内交通

地铁（强烈推荐）：

  • 11条线路覆盖主要景区
  • 票价：¥2–8
  • 站内自助售票机购票（支持英文）
  • 支付宝乘车码通用地铁和公交

出租车/滴滴：

  • 起步价¥11（3公里），之后¥2.9/公里
  • 滴滴App支持英文；市内出行价格合理

步行/骑行：市中心相对紧凑。夫子庙到老门东、中华门之间徒步最佳。共享单车（哈啰/美团）随处可扫——¥1.5/30分钟。

城际交通

南京禄口国际机场（NKG）：地铁S1线直达市区（50分钟，¥8）。出租车到市中心约¥120–150（40分钟）。

高铁：南京是华东重要铁路枢纽。

  • 南京南站：主要高铁站点
  • 至上海：1–1.5小时（¥140–230）
  • 至杭州：1.5小时（¥120–200）
  • 至北京：3.5小时（¥440–750）
  • 至苏州：50分钟（¥100–160）

实用贴士

支付：

  • 支付宝/微信支付全城通用——出发前绑定国际信用卡
  • 中国银行ATM支持国际卡取现；备¥300现金应急

语言：

  • 普通话通用；景区年轻人基础英语尚可
  • 提前下载Pleco词典或Google翻译离线包
  • 把酒店名称和地址写成中文存在手机上

网络：

  • Google、WhatsApp、Facebook均被屏蔽——出发前安装VPN
  • 多数酒店和咖啡馆有免费WiFi（需护照登记）
  • 机场联通旅游SIM卡：¥100/7天不限流量

安全：

  • 南京非常安全——暴力犯罪率低，夜间街道热闹
  • 夫子庙等拥挤景点注意扒手，早晚高峰地铁也需留意
  • 警惕"茶托"骗局：陌生人邀请喝茶，结账时天价账单

礼仪：

  • 南京人出了名的热情直爽——这座城市自带从容淡然的气质
  • 不设小费文化，无需额外付小费
  • 有序排队；相比超大城市，南京人排队更规矩
  • 侵华日军南京大屠杀遇难同胞纪念馆是庄严肃穆之地——衣着得体，低声交谈，不拍照
